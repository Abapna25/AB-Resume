# 📋 Project Management SOPs & Frameworks

A collection of standard operating procedures, governance frameworks, and delivery artifacts authored and implemented in a production enterprise environment — covering a full-stack data platform serving 900+ institutions.

These documents reflect end-to-end ownership: from initial process design through stakeholder alignment, team adoption, and iterative refinement across multiple Agile delivery cycles.

## 🛠 Context & Scope

These SOPs were developed while leading delivery of a cloud-based data platform built on **Power BI**, **Azure Data Factory (ADF)**, and **SQL**, with integrations managed via **Jira** and **Confluence**.

Key areas of ownership:
- Scaled the delivery team from 3 to 10 members while maintaining sprint cadence
- Established Agile processes from scratch, including all ceremonies and documentation standards
- Pioneered an internal AI governance program including a Microsoft Copilot pilot
- Led CMMI appraisal participation and ISO audit readiness
